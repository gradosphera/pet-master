- Рабочая температура блока: °C (рекомендовано 210–215°C).

## 4. Тестовая печать (Параметры слайсера)

Внимание: ПЭТ-филамент из бутылок имеет более высокую температуру плавления и полый профиль.

| Параметр                     | Значение (рекомендуемое) | Значение (факт) |
| ---------------------------- | ------------------------ | --------------- |
| Температура сопла            | 255 – 265 °C (до 300°C)  |                 |
| Температура стола            | 80 – 85 °C               |                 |
| Коэффициент экструзии (Flow) | 130 – 135 %              |                 |
| Скорость печати              | 30 – 50 мм/сек           |                 |
| Обдув модели                 | 0 – 30%                  |                 |

---

## 5. Оценка результатов печати (чек-лист)

- Адгезия: Первый слой стабильно держится на столе.
- Межслойная спекаемость: Деталь монолитная, не расслаивается при нагрузке.
- Поверхность: Отсутствие «пузырей» (свидетельствует о хорошей просушке сырья).
- Цвет: Соответствует исходной таре, без признаков деградации (почернения) полимера.

## Подпись ответственного:

## 💡 Памятка волонтеру:

- Тугоплавкость: Помните, что вторичный ПЭТ более вязкий, чем покупной PETG. Если ваш принтер не поддерживает нагрев до 270-300°C, убедитесь, что термобарьер имеет медную вставку или является цельнометаллическим.
- Поток 135%: Не пугайтесь высокого значения потока. Пултрузионная нить — это «свернутая трубочка», внутри которой есть пустота. Увеличение потока компенсирует этот внутренний объем, делая стенки модели прочными.
